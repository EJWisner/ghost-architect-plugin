# Ghost Architect — Claude Code Plugin

Codebase triage for Adobe Commerce and ecommerce platforms.

Ghost Architect scans your project, categorizes risk by pattern, and gives your team a prioritized map of where to start — before you touch a line of code.

## What it does

- Scans your codebase locally — code never leaves your machine
- Categorizes findings by severity: Critical, High, Medium, Low
- Identifies customization risk, security patterns, upgrade blockers, and technical debt
- Gives your team a starting point, not a wall of noise

## Usage

Inside Claude Code, run /ghost-architect:triage and Claude will scan your current project and return a prioritized findings report.

## Tiers

- Ghost Open: free, included with this plugin
- Ghost Pro: full PDF reports, multipass analysis, project intelligence — ghostarchitect.dev

## Security

- All analysis runs locally
- Anthropic API calls use zero data retention
- Your code is never stored or used for training
